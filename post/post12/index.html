Chrome DevTools - a built-in color picker

Did you know that Chrome DevTools comes with a built-in color picker?

Google Chrome DevTools - a built-in color picker that you probably didn’t know existed.

Chrome DevTools is a set of very useful tools built directly into the browser. Among all these tools, there is also a color picker which this post will be about. Let’s start.

First we need to open Chrome DevTools. You can do this by using keyboard shortcuts:

  • Ctrl + Shift + I (Windows)
  • Command + Option + I (Mac)

or you can just right-click somewhere on the page and then select “Inspect”:

Once you launch the DevTools, just select the element whose color you would like to modify:

Now, in the “Styles” tab we can see that it’s color is #1877f2. Click on the color box. This will present more options for you, as per below screenshot:

Here, you can easily modify it’s color and opacity but also there is a built-in color picking tool, so you can check and copy color of any element you want:
