Pseudo Website Scraper - copy function

We can use the copy() function to scrape data from the webpage. How? Let's find out.

Let’s create a pseudo data scraper with copy() function and browser console.

The copy() function can be used to easily fetch some data from the page. For example, let’s say that we want to fetch the list of all the team members from Our Team gallery that looks like this:

<img src="name_familyname.jpg" alt="Name Familyname" width="500" height="600">

In this case, we can use the copy() function and alt attribute:

copy($$('img').map(img => img.alt).join('\n'))

Let’s see this in action (remember however, that copy() function will copy anything we passed to it as an argument but it won’t display anything in the console. We need to paste the values manually):

Let us see some more examples. For instance, let’s scrape all the URLs found on a page:

copy($$('a').map(a => a.href).join('\n'))

or all the options from a dropdown list:

copy($$('option').map(option => option.text).join('\n'))
