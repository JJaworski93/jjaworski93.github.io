JavaScript Best Practices - Cheatsheet (No. 4)

JavaScript Tips and Best Practices - Cheatsheet

JavaScript - a fourth list of the Tips and Best Practices.

This is the fourth article in the JavaScript Tips & Best Practices series. Let’s get right into it.

16. Merging arrays with spread operator. The Spread Operator (three dots) has many different uses and I will definitely write a separate article about it. Basically, the Spread Operator allows us to spread out elements of an iterable object such as an Array, Set, or Map. Today, however, let us see a merging arrays examples:

const someNumbers = [-11, 0, 15, 188];
const moreNumbers = [-2, 799, 99];

const finalNumbers = [...someNumbers, ...moreNumbers];
console.log(finalNumbers); // [-11, 0, 15, 188, -2, 799, 99]


const someNumbers = [4, 5, 6];
const combinedNumbers = [1, 2, 3, ...someNumbers, 7];

console.log(combinedNumbers); // [1, 2, 3, 4, 5, 6, 7]

17. Getting the last item(s) from an array. To get the last item(s) from an array we can use slice:

let array = [0, 1, 2, 3, 4, 5];

console.log(array.slice(-1)); // [5]
console.log(array.slice(-2)); // [4, 5]
console.log(array.slice(-3)); // [3, 4, 5]

To get the last item from an array we can use also length:

let array = [0, 1, 2, 3, 4, 5];

console.log(array[array.length -1]); // [5]
console.log(array[array.length -2]); // [4]

18. Filtering unique values/Removing duplicates. To filter unique values in JS, we can use Set with Spread Operator:

let myArray = ["abc", 1, 2, 3, "abc", 1, 2, 'a', -5, '1', 'a', 1, 2];
let uniqueValues = [...new Set(myArray)];

console.log(uniqueValues); // ["abc", 1, 2, 3, 'a', -5, '1']

In this case, the constructor of Set takes an iterable object (like an Array) and the Spread Operator (…) transforms the Set back into an Array.

20. Converting an Array to an Object with Spread Operator. To achieve this, we can use our already well-known Spread Operator:

let team = ["Ada", "Ala", "Alex", "Jakob"];
let teamObject = {...team};
console.log(teamObject); // {0: "Ada", 1: "Ala", 2: "Alex", 3: "Jakob"}
