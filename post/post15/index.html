JavaScript Best Practices - Cheatsheet (No. 3)

JavaScript Tips and Best Practices - Cheatsheet

JavaScript - a third list of the Tips and Best Practices.

This is the third article in the JavaScript Tips & Best Practices series. Let’s get straight to the point.

11. Verify that given argument is a number - to verify that a given argument is a number, we could use below function:

function isNumber(n){
    return !isNaN(parseFloat(n)) && isFinite(n);
}

console.log(isNumeric(12345));                // true
console.log(isNumeric('5 '));                 // true
console.log(isNumeric("-55.1 "));             // true
console.log(isNumeric(-55.2));                // true
console.log(isNumeric(-55,2));                // true

console.log(isNumeric("-55,2"));              // false   
console.log(isNumeric("abcc"));               // false      
console.log(isNumeric(''));                   // false
console.log(isNumeric(undefined));            // false
console.log(isNumeric(null));                 // false
console.log(isNumeric([]));                   // false

12. Get max or min from an array of numbers to get the max or the min value from an array of numbers:

let arrayOfNumbers = [5, 508, -5.5, 5800.5, -512, 228, 0.1, 0, 1555505, -8585]; 

let maxNumber = Math.max.apply(Math, arrayOfNumbers);     // 1555505
let minNumber = Math.min.apply(Math, arrayOfNumbers);     // -8585

13. Empty an array. We could achieve this by setting array’s length to 0. This method has been proven to be one of the fastest in terms of performance:

let someArray = [12, 222, 1000, 0.1, 'a', "absd"];  

myArray.length = 0;     // [].

14. Remove an item from an array. It is considered to be a good practice to use slice instead of delete, when removing an item from an array. This is due to the fact that delete (unlike slice) does remove the element, but does not reindex the array or update its length. Let’s see an example of both:

// removing an item with delete:
let someArray = [8, 585, 'a', 5478, 'absd', 0.8, 2154]; 
someArray.length;           // 7
delete someArray[5];
someArray.length;           // 7 
console.log(someArray);     // [8, 585, 'a', 5478, 'absd', empty, 2154]
console.log(someArray[5]);  // undefined

// removing an item with slice:
let someArray = [8, 585, 'a', 5478, 'absd', 0.8, 2154]; 
someArray.length;           // 7
someArray.splice(5,1);
someArray.length;           // 6 
console.log(someArray);     // [8, 585, 'a', 5478, 'absd', 2154]
console.log(someArray[5]);  // 2154

15. Truncate an array. In order to truncate an array we can use length:

let someArray = [8, 585, 'a', 5478, 'absd', 0.8, 2154]; 
someArray.length = 3;
console.log(someArray);     // [8, 585, 'a']
