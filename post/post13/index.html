JavaScript Tips and Best Practices - Cheatsheet

JavaScript - a list of the Tips and Best Practices.

In this article, I decided to gather some of the JavaScript tips and best practices. Let’s get right into it!

1. Truthy / Falsy values - in JavaScript all values, except those listed below, are considered to be truthy:

  • false
  • 0 (but not as a string)
  • ““ and '' (empty strings)
  • null
  • undefined
  • NaN (Not a Number)
