JavaScript Tips and Best Practices - Cheatsheet

JavaScript - a second list of the Tips and Best Practices.

This is the second article in the JavaScript Tips & Best Practices series. Let’s cut straight to the chase.

6. Object Constructor - is a special function that creates and initializes an object instance of a class. The purpose of such constructor is to create a new object and set values for any existing object properties. It is considered a good practice to name constructor functions with an upper-case first letter. Let’s see an example:

function Person(firstName, lastName, age) {
    this.firstName =  firstName;
    this.lastName = lastName; 
    this.age = age;      
}

let Jakub = new Person("Jakub", "Thebugger", 35);

7. Immediately Invoked Function Expression (IIFE). To simplify, we can say that an IIFE is a function that runs as soon as it is defined. Let’s see some examples:

// anonymous IIFE
(function () {
    ...
    ...
})();
// named IIFE
(function someName() {
    ...
    ...
})();
// arrow IIFE
(() => {
    ...
    ...
})();
// async IIFE
(async () {
    ...
    ...
})();

It is considered to be a good practice to name your IIFEs (e.g. for debugging perspective). Why IIFE?

  • protection against pollution of the global environment,
  • simple way to isolate variables declarations,
  • easy solution to make code more organize and maintainable (modular approach).

8. Random Integer. Getting random integer from a specific range (between min and max value):

// function that returns a random integer between two values, excluding max value <min, max):
function randomIntegerBetween(min, max) {  
  return Math.floor(
    Math.random() * (max - min) + min
  );
}
// function that returns a random integer between two values, including max value <min, max>:
function randomIntegerBetween(min, max) {  
  return Math.floor(
    Math.random() * (max - min + 1) + min
  );
}
// assigning a random integer between two values to a variable, e.g. in order to generate fake data for testing purposes:
let randomInteger = Math.floor(Math.random() * (max - min + 1)) + min;

9. Random Array Item. Getting random item from an array:

// IIFE function that returns a random array item:
(function randomArrayItem() {  
    let arrayItems = [8, 512, 'a', 7, 507, 'bcd', 88, 'Thebugger', true];
    return arrayItems[Math.floor(
    Math.random() * arrayItems.length
  )];
})();
// assigning a random array item to a variable:
let arrayItems = [8, 512, 'a', 7, 507, 'bcd', 88, 'Thebugger', true];

let  randomArrayItem = arrayItems[Math.floor(Math.random() * arrayItems.length)];
